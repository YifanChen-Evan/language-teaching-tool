Get Started

Step 1 Load a Local Folder

In "Home" section, you can click "Start Lessons" button to select the folder stored locally as the source of the json file.

How to write your json file

Step 2 Select a Video

Then the json files available in that folder will show up in the "Learning Resource Library". You can see the video title, number of questions and level stored in each json file to choose the learning material that suits you.

Step 3 Start a Lesson

After selecting the appropriate learning video, you will be redirected to the "Lesson" page. Now you can enjoy your custom lesson. While watching the video, you need to complete the quiz questions that pop up at specific timestamp.

Step 4 Get a Quiz Score

Quiz scores will be announced at the end of the video. Note that you will only receive score if you have completed all of the quiz questions in this lesson.
