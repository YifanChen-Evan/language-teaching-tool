Lesson Description File

The lesson description file (LDF) is a file that stores various attributes of the lesson in json format, including video title, video link, video cover, difficulty level and data related to quiz questions.

"title"

It refers to the video title.

"iframeVideoUrl"

It is used to store external links of embedded video.

"videoImage"

You can store the relative/absolute path of the video cover.

"level"

It refers to the overall difficulty level of the quiz questions in the video. Its value can be "Beginner", "Intermediate" or "Advanced".

"jsonData"

It stores 5 properties describing each quiz question.

  • "timestamp"

    It is used to set the time point at which the quiz question pops up. Its value is the time in seconds. For example, to set the first quiz question at 4 minutes and 50 seconds, the value of the "timestamp" key would be 290 (60 * 4 + 50 = 290).

  • "question"

    It is the question of quiz at the specific timestamp.

  • "answers"

    The options for the question are stored in the "answer" as an array.

  • "correctAnswer"

    It is the index of the correct answer in the "answer" array.

  • "gradingWeights"

    The grading weight for each question can be set individually without having to set the same score for all questions.
